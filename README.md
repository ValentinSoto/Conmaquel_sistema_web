## Descripción

Este proyecto desarrolla un Sistema Web de Gestión Integral para la empresa Conmaquel, aplicando la metodología Scrum para optimizar la gestión de productos, reservas, ventas e inventario.

## Objetivos

- Automatizar procesos.
- Mejorar la gestión de reservas.
- Optimizar el control de inventario.
- Generar reportes.
