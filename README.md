## 📖 Descripción

El presente proyecto consiste en el desarrollo de un **Sistema Web de Gestión Integral para la empresa Conmaquel**, utilizando la metodología ágil **Scrum** para optimizar la administración de productos, inventario, reservas y ventas, mejorando la eficiencia de los procesos internos de la organización.

# 🎯 Objetivo General

Desarrollar un Sistema Web de Gestión Integral para la empresa Conmaquel mediante la metodología Scrum, optimizando la gestión de productos, reservas, ventas e inventario.

---

# ✅ Objetivos Específicos

- Analizar los procesos actuales de la empresa.
- Levantar los requerimientos funcionales y no funcionales.
- Diseñar la arquitectura del sistema.
- Modelar la base de datos.
- Aplicar Scrum durante el desarrollo.
- Implementar los módulos del sistema.
- Validar el funcionamiento mediante pruebas.
