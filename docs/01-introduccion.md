# Introducción

## Descripción

La transformación digital ha impulsado a las organizaciones a adoptar soluciones tecnológicas que permitan optimizar sus procesos internos, mejorar la atención al cliente y fortalecer la toma de decisiones. En este contexto, el desarrollo de sistemas de información constituye una herramienta estratégica para incrementar la eficiencia operativa y la competitividad empresarial.

El presente proyecto desarrolla un **Sistema Web de Gestión Integral para la empresa Conmaquel**, aplicando la metodología ágil **Scrum** para planificar, diseñar e implementar una solución informática orientada a la gestión de productos, reservas, ventas e inventario. La utilización de Scrum permite organizar el trabajo en iteraciones cortas, facilitar la colaboración entre los integrantes del equipo y asegurar la entrega continua de valor al cliente.

La propuesta contempla el análisis de los procesos actuales de la empresa, el levantamiento de requerimientos funcionales y no funcionales, el diseño de la arquitectura del sistema, la modelación de la base de datos, la elaboración de diagramas UML y BPMN, así como la planificación de las actividades mediante herramientas propias de Scrum, como el Product Backlog, Sprint Planning, Burndown Chart y Cronograma de Gantt.

Asimismo, el proyecto incorpora principios de Ingeniería de Software y buenas prácticas de desarrollo para garantizar que el sistema sea seguro, escalable, mantenible y de fácil utilización por parte de los usuarios.

---

# Objetivo General

Desarrollar un Sistema Web de Gestión Integral para la empresa Conmaquel mediante la aplicación de la metodología Scrum, con el propósito de optimizar la gestión de productos, inventario, reservas y ventas.

---

# Objetivos Específicos

- Analizar la situación actual de la empresa Conmaquel.
- Identificar los requerimientos funcionales y no funcionales del sistema.
- Diseñar la arquitectura del sistema y el modelo de base de datos.
- Elaborar los diagramas UML y BPMN necesarios para el desarrollo.
- Planificar el proyecto utilizando la metodología Scrum.
- Implementar una solución informática orientada a mejorar la gestión empresarial.
- Evaluar el funcionamiento del sistema mediante pruebas de validación.

---

# Alcance del Proyecto

El proyecto comprende el análisis, diseño y documentación de un sistema web para la gestión integral de la empresa Conmaquel. Incluye la administración de usuarios, productos, inventario, reservas, ventas y generación de reportes, considerando el uso de herramientas y metodologías ágiles para el desarrollo del software.
